Welcome to

In this tool, you can visualize and explore the set of non-dominated optimal variants (called Pareto front) computed by one of the various optimization backends. Press the '?' button of each window for a specific help.
  • Use the view Input Clafer Model and Options to upload, select an example, or enter a Clafer model to optimize.
  • Use the view Objectives and Quality Ranges to observe optimization objectives and filter by quality.
  • Use the views Feature and Quality Matrix, Bubble Front Graph, and Parallel Coordinates Chart to observe the entire Pareto front, to filter by features or quality, and to select a subset of variants.
  • Use the views VariantComparer and Spider Chart to perform tradeoff analysis on selected variants.
  • Use the view Output to see compilation and error messages.
