Parallel Coordinates Chart

In this window, the Pareto front is represented by a set of polylines connecting several parallel axes. The first axis denotes a variant index, the rest axes represent quality attributes. Each polyline is a variant (instance).
  • Mouse over a line to highlight a variant.
  • Filter by quality values by dragging a mouse over a range on a particular axis.
  • Reset filters by clicking at any point on an axis.
  • Click on a line to select or unselect a variant.
  • Swap axes with each other using drap and drop.
